
# Portfolio Mariusz Najwer
[https://najwer23.github.io/](https://najwer23.github.io/)


## Requirements
* Yarn ^1.22.0
* Node ^14.16.0

## Instalation
```
git clone https://github.com/najwer23/najwer23.github.io.git
```
## Run in folder client/
```
yarn install
yarn start
```
## Deploy - Run in folder client/
```
yarn run deploy
```
